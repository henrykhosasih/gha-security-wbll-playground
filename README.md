# BAD Template For Learning How to Harden GitHub Actions

This repository is a sample with **misconfigured** settings and workflows, intended as a teaching aid for locking down Action workflows. It complements my [gasa](https://github.com/BretFisher/gasa) tool that detects common issues.

If you want to see a GOOD example of a hardened GitHub Actions workflow, check out the [gasa-pass](https://github.com/BretFisher/gasa-pass) repository.
